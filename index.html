Welcome to Autotest

Introduction

Autotest is a prototype application investigating methods of web-application test automation, with an emphasis on tests for accessibility.

Implemented tests

  • state: focus and hover states of form controls
  • autocom: autocomplete attributes of inputs
  • role: elements with role attributes, listed
  • roles: elements with role attributes, tabulated
  • imgdec: images marked up as decorative
  • imginf: images marked up as informative
