Welcome to Autotest

Introduction

Autotest is a prototype application investigating methods of web-application test automation, with an emphasis on tests for accessibility.

Your instructions to Autotest are in a script.

A script follows Autotest script rules. Each script is stored in its own file.

The standard place to put your script files is a directory named autotest-scripts next to the autotest directory.

You will tell Autotest where your scripts are located. Then Autotest will show you a list of your scripts. You will choose a script to run from. Then Autotest will perform the actions specified in the script and create a report.
